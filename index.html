A beginner's guide to tables using HTML & CSS.

HTML Tags: Building Your Table

All HTML tables are made up of three basic things: rows, columns, and cells. Let's break down the HTML tags used to create a table and its basic parts.

`
Tag Tag Name Tag Description
<table> Table Opening bracket for making a table in HTML. All table content must be enclosed in <table> and </table>.
<thead> Table Head Contains the table head content. Always use <thead> for the header row instead of the <td> tag.
<tr> Table Row Contains all of our table data. You can use as many rows as you'd like, but all rows must have the same amount of data.
<td> Table Data Contains the content of table cells that make up a table's rows.

CSS: Styling Your Table

CSS (or cascading style sheets) are what give your project life. Without CSS, your table would look incredibly boring and would consist of black text on a white background. Now, there is nothing wrong with that by any means! But why stick to a plain design when you can let your creativity run free!


Property Use With: Example
Table Borders <table>, <th>, <td> .th { 1px solid black }
Full-Width Table <table> .table { width: 100% }
*Single Table Border <table> .table { border-collapse: collapse }
Border Spacing <table> .table { border-spacing: 20px }
*Used to remove double borders in a table by collapsing the borders into a single border.

There are way more styling options than just these few that can allow you to change the design of your table. Some of my most commonly used stylings are: color, background-color, margin, padding, font-family, font-size, position, text-align, text-weight, height, and width.


My best recommendation - make a basic table and just expirment with different styling on a CSS sheet. See how things change when you add or change different values. Find something you like? Jot it down so you will remember it for a future project!
